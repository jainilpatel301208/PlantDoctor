# PlantDoctor
🌿 AI-powered plant identification and disease diagnosis — upload a leaf photo and get instant species ID, disease detection, and treatment tips 🔍

====Plant Doctor AI====

An AI-powered web app that identifies plants and diagnoses leaf diseases from a photo.

Upload a leaf photo (camera, file upload, or image link), and get:


Plant species identification
Disease detection with severity
Treatment suggestions (organic and chemical)
Watering, sunlight, and care guidance

=======Tech=========
HTML / CSS / JavaScript
Dify AI agent for plant diagnosis
